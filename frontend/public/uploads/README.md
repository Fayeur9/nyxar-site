# Structure des uploads

Ce dossier contient toutes les images uploadées par le système.

## Organisation des dossiers

- **categories/** : Images des catégories de vote
- **competitions/** : Images des compétitions
- **games/** : Images des jeux (image principale et image hover pour l'effet 3D)
- **guess-map/** : Images des challenges Guess Map
- **herobanner/** : Images du hero banner (page d'accueil)
- **line_ups/** : Images de couverture des line-ups
- **players/** : Photos des joueurs
- **resultats/** : Images des résultats de compétitions
- **roles/** : Icônes des rôles
- **skins/** : Images des skins (image principale et image hover)
- **sponsors/** : Logos des sponsors
- **users/** : Avatars des utilisateurs

## Accès aux images

Les images sont accessibles via l'URL `/uploads/` depuis le frontend.

Exemple : `/uploads/games/trackmania.jpg`

## Convention de nommage

- Utilisez des noms en minuscules
- Remplacez les espaces par des tirets (-)
- Format recommandé : jpg, png, webp
